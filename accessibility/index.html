Accessibility Statement

Every effort has been made to making every web page on this site accessible to all. I have tried to follow the Web Content Accessibility Guidelines (WCAG) AAA guidelines where possible, or AA at least.

All contrast ratios meet W3's The W3C Web Content Accessibility Guidelines’ (WCAG) “Contrast Enhanced” Level AAA criterion, allowing users with 80:20 vision or better to be able to make out all text

Users of screen readers will be able to understand what is in images by the alt tags in the HTML of the pages.

All styling is done through CSS rather than JavaScript, so custom stylesheets can be applied to accommodate any user's custom visual needs.
