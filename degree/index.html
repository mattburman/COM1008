Web and Internet Technology

View module page

This module focuses on networks, infosec, and front-end web development.

There is also a focus on web graphics: both SVG and canvas. The canvas is used within the project assignment to create a web-based game.

Factfile

  • Assessment: 80% assignment; 20% quizzes (10% each)
  • Teaching methods: Lectures supplemented by relevant labs
  • Lecturer: The one and only Dr. Steve Maddock

Assessment - very practical

60% of the assessment is on creating a website utilising best responsive web design practices. 10% of the module is for creation of a design document, and another 10% is for a development and testing document.

There are also two quizzes at 10% each. Due to the very practical nature of web design, it is great that 80% of this module is assessed via assignment rather than exams or quizzes.

Introduction to Software Engineering

View module page

This module introduces key software engineering principles.

Software lifecycles; Version control (mostly Git); Project management; Design principles; Metrics and estimation; and the legal and ethics of software.

Ruby is also taught alongside these topics, making use of git when programming, preparing us for the second semester.

The first semester has a group project where a client has a software idea which they want implementing. As a group, we must take their idea, and create user stories ready for second semester's implementation.

Factfile

  • Assessment: Quizzes, assignments, team project
  • Teaching methods: Lectures supplemented by relevant labs
  • Lecturer: Dr. Gordon Fraser

Foundations of Computer Science

View module page

This module introduces key mathematical concepts every computer scientist should know. Propositional logic and set theory; predicate logic; formal proofs; proof by induction; functions and relations.

Factfile

  • Assessment: 25% from 5 problem sheets; 75% from 3 quizzes
  • Teaching methods: Lectures with labs
  • Lecturer: Dr. Paul Watton

Java Programming

View module page

This module is an introduction to programming, starting at the very basic, making it’s way up to object oriented programming concepts towards the end of the semester.

There are 4 quizzes and 3 assignments making up the marks for the semester. The third assignment is a group project where we are tasked with programming a LEGO Mindstorm robot to find it’s way around a known, deterministic, static, partially observable, single-agent, episodic, and continuous task environment.

Factfile

  • Assessment: 48% over 4 quizzes (12% each); 52% over 3 assignments (12%, 20%, 20%)
  • Teaching methods: Lectures supplemented by relevant labs
  • Lecturer: Siobhán North

Machines and Intelligence

View module page

This module is an overview of the field of artificial intelligence. Starting with an in-depth look at the history of AI, it then quickly looks at subfields such as machine learning, logical reasoning and agents, computer vision, natural language processing, and robotics.

There are lab sessions in which we have been controlling NAO Robots. We used their speech recognition and computer vision capabilities to get it recognise our faces and say hello to us, for example.

Factfile

  • Teaching methods: Lectures with labs
  • Lecturer: Prof. Rob Gaizauskas

Devices and Networks

View module page

This module starts off with an overview of low level computing systems, starting with binary representations of data in a computer, moving on to logic circuits, making it’s way up to instruction set architectures.

This was an optional module which I chose with my free 20 credits.

Factfile

  • Assessment: 15% of the whole year is by 5 multiple choice quizzes in the first semester.
  • Teaching methods: Lectures supplemented by relevant labs
  • Lecturer: Dr. Dirk Sudholt
